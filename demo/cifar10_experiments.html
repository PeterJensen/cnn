CIFAR-10 Experiments

These experiments are using a modified version of the amazing convnet.js framework. This framework enables users to define a multi layer neural network. Features for both training and inference are available for the custom network. I've experimented with the cifar10 data set for the experiments below.

Using Web Workers

This experiment uses Web Workers for the forward computation of the convolution layers in a pre-trained network. The various layers form a pipeline, where each convolution stage in the pipeline has a dedicated Web Worker. Performance profiling reveals that the convolution layers are responsible for the majority of the compute time required. By using Web Workers the per image processing time is cut roughly in half.
